Parágrafos e Quebras de linha

A tag "hr" <hr> cria uma linha horizontal. Ela não tem tag de fechamento. Caso seja inserida no meio do texto de um parágrafo, ela cria a linha no local onde é inserida e automaticamente cria uma quebra de linha.

A tag "br" (break roll) <br> é utilizada para quebrar a linha, até mesmo dentro de um parágrafo. Esta tag não tem fechamento, somente abertura:
